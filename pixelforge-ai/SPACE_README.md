# PixelForge AI

2D 游戏素材生成器 — 文本描述 → AI 生成 → 引擎导出

## 功能

- 精灵生成: 角色、道具、NPC
- 瓦片生成: 地形、建筑、场景 (4主题 + autotile)
- UI 生成: 按钮、面板、图标
- 特效生成: 粒子、爆炸、魔法
- 动画生成: 5种预设动画序列
- 三引擎导出: Unity / Godot / Cocos

## 使用方法

1. 在左侧控制面板选择风格、模板
2. 输入素材描述（支持中文）
3. 点击生成
4. 选择导出引擎 → 下载资源包

## 环境变量

需要在 Space Settings 中设置:
- `SILICONFLOW_API_KEY`: 硅基流动 API Key
